Robin

1-week UI prototype in HTML/CSS/JS that imagines the quest log of a modern detective game adaptation of Robin Hood by J. Walker McSpadden.

I constrained myself to three public domain assets provided for a class assignment: a world map texture, a set of roadmap icons, and a .txt file of the book, which I adapted to suit the project.

Project is best viewed in Google Chrome.

Font used is Cairo.
